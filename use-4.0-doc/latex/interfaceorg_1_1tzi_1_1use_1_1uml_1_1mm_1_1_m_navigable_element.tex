Returns the {\ttfamily Type} of the navigation from {\ttfamily source\-Object\-Type} via the association end {\ttfamily src} to this end.

{\ttfamily src} is required because redefined ends or the navigation over an qualified association without qualifier values can change the type of the navigation.

let C be the class at dst\-End, then the result type is\-:  {\ttfamily C} if the multiplicity is max. one, this is binary association and qualifier values are provided for a qualified association  {\ttfamily Ordered\-Set(\-C)} if this association end is marked as \{ordered\} and qualifier values are provided for a qualified association  {\ttfamily Set(\-C)} if the multiplicity is greater than 1 qualifier values are provided for a qualified association  {\ttfamily Bag(\-C)} if {\ttfamily src} has qualifiers, but no qualifier values are provided  {\ttfamily Sequence(\-C)} if this association end is marked as \{ordered\} and {\ttfamily src} has qualifiers, but no qualifier values are provided


\begin{DoxyParams}{Parameters}

\hline

\hline
{\em qualified\-Access} & Boolean value whether qualifier values are used or not. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
The result {\ttfamily Type} of a navigation with the specified parameters.
