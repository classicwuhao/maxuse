This operation handles filenames provided to the U\-S\-E-\/\-Shell. Surrounding characters like {\ttfamily '} or {\ttfamily "} (even if mixed) are removed. 

If an absolute path is given as {\ttfamily filename}, this file name (with removed quotes) is returned.

If a relative path is provided as {\ttfamily filename}, the currently opened file is used as the starting point to calculate the absolute path.

{\bfseries Warning\-:} No check is made if the calculated file exists. This has to be done by the caller.


\begin{DoxyParams}{Parameters}
{\em filename} & A absolute or relative filename to open. \\
\hline
{\em use\-As\-Current\-File} & If {\ttfamily true}, the opened file is stored as currently opened and is used as the starting point to calculate subsequent relative file names.
