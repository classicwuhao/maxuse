\subsection{Detailed Description}
A \hyperlink{classorg_1_1tzi_1_1use_1_1gui_1_1main_1_1_model_browser}{Model\-Browser} provides a tree view of classes, associations, and constraints in a model. The definition of a selected element is shown in an H\-T\-M\-L pane. A class can be dragged onto an object diagram to create a new object of this class.
