\subsection{Detailed Description}
Test U\-S\-E\-Compiler class. The Test\-Driver reads all files with extension \char`\"{}.\-use\char`\"{} and processes them with the parser. If a file {\ttfamily t.\-use} contains expected errors there should be a file {\ttfamily t.\-fail} with the expected output messages. There are four possible results\-: 
\begin{DoxyEnumerate}
\item file parses ok and no failure file exists\-: P\-A\-S\-S\-E\-D. 
\item file parses ok and a failure file exists\-: F\-A\-I\-L\-U\-R\-E. 
\item file does not parse and a failure file exists\-: P\-A\-S\-S\-E\-D if the actual output matches the expected output from the failure file, otherwise F\-A\-I\-L\-U\-R\-E. 
\item file does not parse and no failure file exists\-: F\-A\-I\-L\-U\-R\-E. 
\end{DoxyEnumerate}
