\subsection{Detailed Description}
A \hyperlink{classorg_1_1tzi_1_1use_1_1util_1_1_report}{Report} formats data as a table. The complete report is built up in memory before it can be printed. This is necessary to determine column widths. Each column may be left-\/aligned, centered, or right-\/aligned.

Creates a new report with the specified number of columns. The format string defines the layout of each row when the report is printed. The position of a column in the format string is specified by a {\ttfamily \$} sign followed by either a {\ttfamily l} character for a left-\/aligned column, a {\ttfamily c} character for a centered column, or a {\ttfamily r} character for a right-\/aligned column. All other characters are copied verbatim into the output. If there are more format specifiers than columns, the excess specifiers are ignored. If there are fewer format specifiers than columns, all columns with no format specifier are left-\/aligned.

Example\-: 
\begin{DoxyPre}
\hyperlink{classorg_1_1tzi_1_1use_1_1util_1_1_report}{Report} r = new \hyperlink{classorg_1_1tzi_1_1use_1_1util_1_1_report}{Report}(4, "[ $l = $r, $r, $c ]");
\end{DoxyPre}
 A possible result is\-: 
\begin{DoxyPre}
[  foo   =      3,          1.2, false ]
[ foobar = 453453, -1.245345345, true  ]
[  line  =    555,        999.0, true  ]
\end{DoxyPre}
