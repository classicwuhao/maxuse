\subsection{Detailed Description}
Provides methods to generate an evaluable \hyperlink{}{Soil statement} from some text input. 

This is accomplished with the A\-N\-T\-L\-R generated parser and lexer, which build an \hyperlink{}{abstract syntax tree} as an intermediate data structure.
