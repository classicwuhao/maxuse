\subsection{Detailed Description}
An iterator which iterates over generated partitions of two lists. One of the lists is specified as unique.

Each returned partition contains only combinations from one element of the unique list e.\-g.\-:\par
 {\ttfamily \{a,b\} \{c,d\}} (second list is marked as unique) results in\-:\par
 {\ttfamily 0 = \{(a,c), (a,d)\}, 1 = \{(b,c), (b,d)\}}
