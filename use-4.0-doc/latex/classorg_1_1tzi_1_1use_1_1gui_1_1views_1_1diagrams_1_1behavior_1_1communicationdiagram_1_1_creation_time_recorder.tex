\subsection{Detailed Description}
Saves the time stamps of the creation events of objects and links that are treaded as equal by U\-S\-E. This is for all events in the current use session. For example\-: 1 !create ada\-:\hyperlink{class_person}{Person} (ada\-:\hyperlink{class_person}{Person} -\/$>$ \{1\}) 2 !set ... (ada\-:\hyperlink{class_person}{Person} -\/$>$ \{1\}) 3 !destroy ada () 4 !create ada\-:\hyperlink{class_person}{Person} (ada\-:\hyperlink{class_person}{Person} -\/$>$ \{1,4\}) This allows to identify the correct object node if both ada objects are shown in the communication diagram.

The creation time can be used by the raised events. \begin{DoxyAuthor}{Author}
