\subsection{Detailed Description}
Holds variables and their values 

Data is organized as follows\-:\par
 Variables and their respective values are hold as a mapping
\begin{DoxyCode}
String 
\end{DoxyCode}
 -\/$>$
\begin{DoxyCode}
Value 
\end{DoxyCode}
 . Mappings for one or more variables are contained in a frame. One or more frames are contained in a level. 

example\-: 

l(n+1) is more recent than l(n), f(n+1) is more recent than f(n) 

l2\mbox{[} f0\mbox{[}x-\/$>$17\mbox{]} \mbox{]}\par
 l1\mbox{[} f0\mbox{[}a-\/$>$'a', x-\/$>$'x'\mbox{]}, f1\mbox{[}a-\/$>$'b'\mbox{]} \mbox{]}\par
 l0\mbox{[} f0\mbox{[}x-\/$>$12, y-\/$>$'y'\mbox{]}, f1\mbox{[}z-\/$>$3\mbox{]} \mbox{]}\par
 

Looking up values only considers the most recent level, and searches backwards from the most recent frame in that level until the value is found or the least recent frame has been searched.\par
 When assigning a value to a variable, a pre-\/existing mapping is searched in the same way a looking up values work. If no such mapping exists, a new mapping is added to the most recent frame in the most recent level.
