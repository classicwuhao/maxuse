\subsection{Detailed Description}
A M\-Attribute can be defined in a base class. To allow a more detailed coverage analysis this class saves the source class of the object expression, e. g., for an access to the name of an professor the class Professor is saved instead of the attribute owner \hyperlink{class_person}{Person} \begin{DoxyAuthor}{Author}
