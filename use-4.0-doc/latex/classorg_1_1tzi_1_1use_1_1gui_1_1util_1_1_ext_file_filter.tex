\subsection{Detailed Description}
A convenience implementation of File\-Filter that filters out all files except for those type extensions that it knows about.

Extensions are of the type \char`\"{}.\-foo\char`\"{}, which is typically found on Windows and Unix boxes, but not on Macinthosh. Case is ignored.

Example -\/ create a new filter that filerts out all files but gif and jpg image files\-: \begin{DoxyVerb}JFileChooser chooser = new JFileChooser();
ExampleFileFilter filter = new ExampleFileFilter(
              new String {"gif", "jpg"}, "JPEG & GIF Images")
chooser.addChoosableFileFilter(filter);
chooser.showOpenDialog(this);
\end{DoxyVerb}
