Detailed Description

A position strategy handles the placement of a node. These strategies can be completely manual like the identity strategy (StrategyIdentity or a combination of automatic values and user information from dragging elements from their automatic position to a user defined one.

This dragging information can be used if the calculated position of a node changes. The most simple case is to move the node by the previously dragged deltas, like it is done in the strategy StrategyInBetween.
