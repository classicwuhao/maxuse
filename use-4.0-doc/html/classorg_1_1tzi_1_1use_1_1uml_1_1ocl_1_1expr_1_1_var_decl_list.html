void org.tzi.use.uml.ocl.expr.VarDeclList.add ( VarDecl  varDecl)
inline

Adds a variable declaration to the list. The following constraints are enforced: (1) The declaration to be added must have a variable name which is distinct from all currently existing declarations in the list. (2) The new declaration must have the same type as all currently existing declarations if the list has been constructed with the allHaveSameType flag.

Exceptions
IllegalArgumentExceptionone of the constraints above failed.
