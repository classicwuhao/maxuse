Type org.tzi.use.uml.mm.MAssociationEnd.getType ( Type  sourceObjectType,
MNavigableElement  src,
boolean  qualifiedAccess 
)
inline

Returns the Type of the navigation from sourceObjectType via the association end src to this end.

src is required because redefined ends or the navigation over an qualified association without qualifier values can change the type of the navigation.

let C be the class at dstEnd, then the result type is: C if the multiplicity is max. one, this is binary association and qualifier values are provided for a qualified association OrderedSet(C) if this association end is marked as {ordered} and qualifier values are provided for a qualified association Set(C) if the multiplicity is greater than 1 qualifier values are provided for a qualified association Bag(C) if src has qualifiers, but no qualifier values are provided Sequence(C) if this association end is marked as {ordered} and src has qualifiers, but no qualifier values are provided

Parameters
sourceObjectTypeThe concrete Type the navigation starts from.
srcThe source end the navigation starts from.
qualifiedAccessBoolean value whether qualifier values are used or not.
Returns
The result Type of a navigation with the specified parameters.
