void org.tzi.use.uml.mm.MClassImpl.addAttribute ( MAttribute  attr) throws MInvalidModelException
inline

Adds an attribute. The attribute must have a unique name within the class and must not conflict with any attribute with the same name defined in a superclass or subclass or with a rolename of an association this class participates in.

Exceptions
MInvalidModelExceptionthe class already contains an attribute with the same name or a name clash occured.
