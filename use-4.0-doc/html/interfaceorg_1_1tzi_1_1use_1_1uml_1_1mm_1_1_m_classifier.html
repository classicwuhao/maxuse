boolean org.tzi.use.uml.mm.MClassifier.isAbstract ( )

If true, the Classifier does not provide a complete declaration and can typically not be instantiated. An abstract classifier is intended to be used by other classifiers (e.g., as the target of general metarelationships or generalization relationships). Default value is false. [UML 2.3, p. 53]

Returns
