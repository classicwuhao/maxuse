List<MNavigableElement> org.tzi.use.uml.mm.MAssociation.navigableEndsFrom ( MClass  cls)

Returns a list of association ends which can be reached by navigation from the given class. Examples:

  • For an association R(A,B), R.navigableEndsFrom(A) results in (B).

  • For an association R(A,A), R.navigableEndsFrom(A) results in (A,A).

  • For an association R(A,B,C), R.navigableEndsFrom(A) results in (B,C).

  • For an association R(A,A,B), R.navigableEndsFrom(A) results in (A,A,B).

This operation does not consider associations in which parents of cls take part.

Returns
List(MAssociationEnd)
Exceptions
IllegalArgumentExceptioncls is not part of this association.
