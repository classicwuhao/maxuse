String org.tzi.use.main.shell.Shell.getFilenameToOpen ( String  filename,
boolean  useAsCurrentFile 
)
inline

This operation handles filenames provided to the USE-Shell. Surrounding characters like ' or " (even if mixed) are removed.

If an absolute path is given as filename, this file name (with removed quotes) is returned.

If a relative path is provided as filename, the currently opened file is used as the starting point to calculate the absolute path.

Warning: No check is made if the calculated file exists. This has to be done by the caller.

Parameters
filenameA absolute or relative filename to open.
useAsCurrentFileIf true, the opened file is stored as currently opened and is used as the starting point to calculate subsequent relative file names. After the file is no longer the current file, i.e., it was closed or the file was not opened, setFileClosed() must be called.
Returns
