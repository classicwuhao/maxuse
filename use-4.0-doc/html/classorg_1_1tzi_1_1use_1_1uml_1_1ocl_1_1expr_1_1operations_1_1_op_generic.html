Detailed Description

OpGeneric is the base class of a large group of individual operations. Each operation is implemented by its own class deriving from OpGeneric. New Operations can easily be added by writing a new operation class and adding a single instance of the new class to the static list of operations in Class ExpOperation (see below). Also, this way the new operation symbol is immediately available to the specification compiler.
