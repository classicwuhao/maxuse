PPCHandler org.tzi.use.uml.sys.MOperationCall.getDefaultPPCHandler ( )
inline

Returns the default PPCHandler of this operation call. The default values are:

  • ExpressionPPCHandler#getDefaultOutputHandler():
    If the called operation has an OCL-body, i. e., it is a OCL query operation.
  • SoilPPCHandler#getDefaultOutputHandler(): If the called operation has a SOIL-body.
  • OpEnterOpExitPPCHandler#getDefaultOutputHandler(): If it has no body (legacy openter / opexit).
  • DoNothingPPCHandler#getInstance(): otherwise.
Returns
The default PPCHandler for the called operation.
