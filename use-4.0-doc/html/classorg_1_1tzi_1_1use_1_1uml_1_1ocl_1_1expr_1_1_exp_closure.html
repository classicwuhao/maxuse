Determines the result type of the closure operation. If source is ordered a ordered set is returned, otherwise a set. The element type of the resulting collection type is retrieved from the range expression.

Parameters
rangeExpThe expression the closure operation is evaluated on.
queryExpThe body of the query expression.
Returns
The result type of this expression.
