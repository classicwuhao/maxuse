void org.tzi.use.uml.mm.MClass.addAttribute ( MAttribute  attr) throws MInvalidModelException

Adds an attribute. The attribute must have a unique name within the class and must not conflict with any attribute with the same name defined in a superclass or subclass or with a rolename of an association this class participates in.

Exceptions
MInvalidModelExceptionthe class already contains an attribute with the same name or a name clash occured.

MOperation org.tzi.use.uml.mm.MClass.operation ( String  name,
boolean  searchInherited 
)

Gets an operation by name. Operations are also looked up in superclasses if searchInherited is true. This method walks up the generalization hierarchy and selects the first matching operation. Thus, if an operation is redefined, this method returns the most specific one.

Returns
null if operation does not exist.
